About This Tool

Images generated by Gemini have a transparent watermark in the bottom right corner. This tool restores the original pixels using a pre-extracted alpha channel and the reverse blending formula:

Original = (Watermarked - Alpha * Logo) / (1 - Alpha)

The tool supports two watermark sizes:

  • 48x48 (for smaller images)
  • 96x96 (for images larger than 1024px)
